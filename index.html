⚙️ Настройка уведомлений в Telegram

Выполните один раз — и каждая заявка будет приходить прямо в ваш телефон

1

Создайте Telegram-бота

Откройте Telegram → найдите @BotFather → напишите /newbot → придумайте название (например: Atlant Leads) и username (например: atlant_leads_bot). BotFather пришлёт вам токен — длинную строку вида 123456:ABC-DEF...

2

Получите ваш Chat ID

Напишите любое сообщение вашему новому боту (нажмите Start). Затем откройте в браузере: https://api.telegram.org/botВАШ_ТОКЕН/getUpdates — найдите в ответе поле "id" внутри "chat" — это ваш Chat ID.

3

Вставьте токен и Chat ID ниже

Нажмите «Сохранить» — готово. Отправьте тестовую заявку чтобы проверить.
